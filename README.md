#TICK TICK
A task management app to help you stay organized and manage your day-to-day. 

#TECHNOLOGIES USED:
REACT
BOOTSTRAP
APOLLO SERVER
GRAPHQL
NODE
MONGODB

#HOW TO RUN?
##for server
cd apollo-server-client
cd server
npm install
nodemon index

##for front-end
cd apollo-server-client
cd client
npm install
npm start
